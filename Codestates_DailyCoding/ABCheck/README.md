## ABCheck

### 문제

문자열을 입력받아 문자열 내에 아래 중 하나가 존재하는지 여부를 리턴해야 합니다.<br>
<br>
1. 'a'로 시작해서 'b'로 끝나는 길이 5의 문자열<br>
2. 'b'로 시작해서 'a'로 끝나는 길이 5의 문자열<br>

### 입력

- string 타입의 알파벳 문자열<br>

### 출력

- boolean 타입을 리턴해야 합니다.<br>

### 주의 사항

- 대소문자를 구분하지 않습니다.<br>
- 공백도 한 글자로 취급합니다.<br>
- 'a'와 'b'는 중복해서 등장할 수 있습니다.<br>
